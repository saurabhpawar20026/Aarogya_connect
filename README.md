# Aarogya Connect

Aarogya Connect is a modern healthcare management web application designed to simplify patient appointment booking, doctor scheduling, and real-time healthcare tracking through an easy-to-use and responsive interface.

---

# 🚀 Features

- 👨‍⚕️ Doctor Appointment Booking
- 📅 Real-Time Appointment Tracking
- 🏥 Patient & Doctor Management
- 🔔 Appointment Notifications
- 📱 Fully Responsive User Interface
- 🔐 Secure Authentication System
- 📊 Easy Healthcare Service Management
- ⚡ Fast and User-Friendly Experience

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
